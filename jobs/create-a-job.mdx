## Overview

Creating a job in CandidHR takes you through a 3-step wizard:
1. **Job Details** — title, type, location, description, skills, salary, and screening settings
2. **Application Form** — configure what candidates fill out and set scoring criteria
3. **Team Members** — optionally invite colleagues to collaborate on this job

Assigning a workflow to the job is **entirely optional** and can be done at any time after the job is created — it is not part of the creation wizard.

## Step-by-step

<Steps>
  <Step title="Navigate to Jobs">
    Click **Jobs** in the left sidebar. You'll land on the jobs list page.

    <Frame>
      <img src="/images/jobs/Jobspage.png" alt="Jobs list page with + New Job button" />
    </Frame>
  </Step>

  <Step title="Click + New Job">
    Click the **+ New Job** button in the top-right corner. This opens the job creation wizard.
  </Step>

  <Step title="Fill in Job Details">
    Complete the job information fields:

    <Frame>
      <img src="/images/jobs/createjobsection.png" alt="Create job form" />
    </Frame>

    | Field | Required | Notes |
    |---|---|---|
    | Job Title | Yes | e.g. "Senior Frontend Engineer" |
    | Employment Type | Yes | Full-time, Part-time, Contract, Internship |
    | Location | Yes | City, state, or "Remote" |
    | Job Description | Yes | Supports rich text — bullets, bold, headings. Use AI Assist to generate or rewrite. |
    | Skills | No | Add required skills and assign resume screening weights |
    | Experience Range | No | Min/max years of experience for resume scoring |
    | Education Requirements | No | Minimum qualification level |
    | Salary Range | No | Shown on the public job page |
    | Close Date | No | When the job stops accepting applications |

    <Tip>
      Use **AI Assist** to auto-generate a job description or extract skills from an existing description.
    </Tip>

    Click **Next** to proceed.
  </Step>

  <Step title="Configure the Application Form">
    Set up what candidates see and submit when they apply.
    <Frame>
      <img src="/images/jobs/Application-form-preferances.png" alt="Application-form-preferances" />
    </Frame>

    **Application preferences** — toggle which fields to show candidates:
    - Work experience, notice period, current CTC
    - Profile picture, intro video
    - GitHub, LinkedIn, personal website

    **Filter criteria** — define hard filters to automatically flag or exclude candidates based on specific criteria.

    **Screening questions** — add custom questions candidates must answer:
    - **Text** — free-form answer
    - **Multiple choice** — single or multi-select
    - **Yes/No** — boolean question
    - **File upload** — e.g. portfolio or cover letter

    You can mark any question as **Required** or **Knockout** (a wrong answer disqualifies the candidate automatically).

    <Tip>
      All application form settings are optional. You can skip this step and configure the form later by editing the job.
    </Tip>

    Click **Next** to proceed.
  </Step>

  <Step title="Invite Team Members (Optional)">
    Add colleagues who should have access to view and manage this job.

    - Search for team members by name
    - Selected members will appear in the list below the search
    - Remove any member by clicking the trash icon next to their name

    <Note>
      This step is optional — you can skip it and add team members later from the job overview page.
    </Note>

    Click **Complete** to finish creating the job.

    <Frame>
      <img src="/images/jobs/jobcreatedsucessfully.png" alt="Job created successfully screen" />
    </Frame>
  </Step>
</Steps>

## Assigning a workflow (optional)

A workflow defines the screening stages candidates move through for this job (e.g. Resume Review → Phone Screen → Interview). Assigning a workflow is **not required** to create a job — you can do it at any time.

To assign a workflow after creating the job:

1. Open the job from the Jobs list
2. Go to the **Workflow** tab on the job detail page
3. Click **Assign Workflow**
4. Select an existing workflow from the dropdown, or create a new one
5. Choose how candidates enter the workflow — via the application form, via email invite, or both

<Note>
  You can change or reassign the workflow at any time, as long as no candidates have progressed past the first stage.
</Note>

See [Build a Workflow](/jobs/build-a-workflow) for a full guide on creating workflows.

## Editing a job after creation

You can edit any job detail at any time:

1. Go to **Jobs** → click the job title
2. Click the **Edit** button on the job overview page
3. Make your changes and save

<Warning>
  Changing the job description or preferences after candidates have applied will not retroactively change their screening scores. Re-score from the Resume Screening tab if needed.
</Warning>

---

## Duplicating a job

To reuse a job setup for a similar role:

1. Open the job from the Jobs list
2. Click the **...** (more options) menu
3. Select **Duplicate Job**

This copies all settings (preferences, questions, workflow assignment) into a new draft job.
