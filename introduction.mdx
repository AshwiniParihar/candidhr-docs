## What is CandidHR?

CandidHR is an end-to-end recruitment platform that helps hiring teams manage the full candidate journey — from job posting to final decision — in one place.

## Key Features

| Feature | Description |
|---|---|
| **Jobs** | Create and manage job postings with custom screening workflows |
| **Applicants** | Centralised table with filtering, bulk actions, and profile views |
| **Assessments** | Aptitude, coding, and custom question banks |
| **Video Interviews** | AI-powered automated video interview with response summaries |
| **Resume Screening** | Automatic resume parsing and weighted scoring |
| **Deep Search** | Scout and invite candidates directly from the platform |
