## Overview

The Analytics Dashboard provides a centralized view of your recruitment activities, candidate engagement, application progress, and feature utilization in real time.

<Steps>
  <Step title="Open the Dashboard">
    Log in and click **Dashboard** from the left-side navigation menu.
  </Step>
  <Step title="Filter by Job">
    Use the **Job Title** dropdown in the top-right corner to filter all metrics by a specific job posting.
  </Step>
</Steps>

## Dashboard Metrics

| Metric | Description |
|---|---|
| **Total Applications** | Total applications received across all job postings |
| **Total Jobs** | Total active or created jobs in the platform |
| **Applicant to Assessment Ratio** | Percentage of applicants who progressed to the assessment stage |
| **Applicant to Interview Ratio** | Number of applicants shortlisted for interviews |
| **Days to Fill** | Average number of days required to close a job position |
| **Job Views** | How many times job postings have been viewed by candidates |
| **Active Candidates** | Candidates currently active in the recruitment process |
| **Drop-off Rate** | Percentage of candidates who discontinued the process |

## Reports & Analytics

| Report | Description |
|---|---|
| **Applications Per Stage** | Stage-wise distribution of all received applications — Resume Screening, Assessments, Video Interviews, Rejected, On Hold, Hired, Final Round |
| **Feature Consumption** | Tracks how often each feature (Resume Screening, Assessment, Video Interview) is accessed |

**Best Practices**
- Use the Job Title filter to analyse performance for individual roles
- Regularly review the Drop-off Rate to identify and fix bottlenecks in the hiring pipeline
- Monitor Active Candidates daily to maintain hiring momentum
- Use the Days to Fill metric to benchmark and optimise your recruitment process
