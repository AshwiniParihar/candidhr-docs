## Overview

The Automated Video Interview module lets you invite candidates to record structured video responses asynchronously. AI scoring provides an immediate performance preview before your manual review.

<Note>
  Only users with **Admin** and **Editor** roles have permission to invite candidates for video interviews.
</Note>

## Dashboard Status Indicators

| Status | Description |
|---|---|
| **Assigned** | The invitation email has been sent |
| **Started** | The candidate has opened the link or is recording |
| **Completed** | The video is ready for your review |
| **AI Score** | Immediate AI-driven performance preview before watching |

<Note>
  AI-generated scores are intended to support recruiter decision-making and should not be considered the sole basis for candidate evaluation.
</Note>

**Best Practices**
- Regularly monitor the Status column to track candidate progress
- Prioritise reviewing candidates marked as Completed to maintain hiring momentum
- Use AI Scores as an initial reference but always review candidate responses manually
- Monitor inactive interviews where status remains Assigned for extended periods

## Inviting Candidates (3-Step Process)

<Frame>
  <img src="/images/video-interviews/image65.png" alt="Step 1 - Select applicants" />
</Frame>

### Step 1: Select Applicants

- Use the **Filter by Job** dropdown to find applicants for a specific role
- For external candidates, enter their email in the **Invite via Email** field
- Toggle the **Requirement** option to require a fresh resume upload before starting

<Frame>
  <img src="/images/video-interviews/image66.png" alt="Step 2 - Select questions" />
</Frame>

### Step 2: Select Questions

- Pull questions from your library or create new ones for this batch
- Monitor the **Selected Questions** count to keep interview length appropriate

<Frame>
  <img src="/images/video-interviews/image67.png" alt="Step 3 - Configuration" />
</Frame>

### Step 3: Configuration

- Set the **Valid Till** date to create urgency and keep the pipeline moving
- Click **Send Interview Link** - CandidHR handles all emails and tracking automatically

<Tip>
  Once the interview invitation is sent, candidates immediately receive access based on the configured settings and deadlines.
</Tip>

**Best Practices**
- Verify candidate email addresses before sending invitations
- Use job-based filters to avoid inviting the wrong applicants
- Keep interview question sets concise and relevant to the role
- Set realistic expiry dates to balance urgency and candidate convenience
- Review all configurations carefully before clicking Send Interview Link

## The Candidate Experience

| Stage | What Happens |
|---|---|
| **Invitation Email** | Clean email with a Start Automated Video Interview button |
| **Browser Requirement** | Candidates are advised to use Microsoft Edge on Desktop/Laptop |
| **Guidelines Page** | Preparation page with full instructions before the timer starts |
| **Tech Check** | Candidates must test their Webcam and Microphone before recording |
| **UI Tour** | Optional tour walks candidates through the interface |

<Note>
  The guided tour is intended to help candidates understand the platform interface and does not affect the actual interview assessment.
</Note>

**Best Practices**
- Encourage first-time candidates to use the guided tour feature
- Use the walkthrough to familiarise candidates with the interface and recording flow
