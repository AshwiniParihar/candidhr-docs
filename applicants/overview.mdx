## Overview

The Applicants section gives you a centralised view of every candidate across all your job postings. You can filter, search, sort, and take action on applicants without leaving the list.

Navigate to **Applicants** from the left sidebar to get started.

---

## Quick Stats

At the top of the page, four summary cards give you an at-a-glance view of your pipeline:

| Card | Description |
|---|---|
| **Total** | Total number of applicants across all jobs |
| **Shortlisted** | Candidates currently marked as Shortlisted |
| **Interviews** | Candidates with an Interview Scheduled status |
| **Offers** | Candidates with an active offer (Offer Extended or Offer Accepted) |

---

## Applicants Table

| Column | Description |
|---|---|
| **Name** | Candidate name - click to open the full profile |
| **Applied For** | Job title(s) the candidate applied to |
| **Status** | Current application status (e.g. Applied, Shortlisted, Hired) |
| **Stage** | Current pipeline stage (e.g. Resume Screening, Assessment) |
| **Approved** | Stage-level approval indicator |
| **Applied On** | Date the application was submitted (sortable) |
| **Last Updated** | Date of the most recent update (sortable) |
| **Action** | Context menu for quick actions |

---

## Filters

Use the filter bar to narrow down the applicant list:

| Filter | Description |
|---|---|
| **Search** | Search by name, email, job title, or stage |
| **Status** | Filter by application status |
| **Stage** | Filter by pipeline stage (Applied, Resume Screening, Assessment, Video Interview) |
| **Job** | Filter by job title |
| **Date Range** | From / To date picker for application date |
| **Only My Applicants** | Show only applicants assigned to you |
| **Shared with Me** | Show only applicants that have been shared with you |

---

## Application Statuses

| Status | Meaning |
|---|---|
| **Applied** | Candidate has submitted their application |
| **In Progress** | Application is actively being reviewed |
| **Shortlisted** | Candidate has been shortlisted for next steps |
| **On Hold** | Decision is paused |
| **Interview Scheduled** | An interview has been scheduled |
| **Final Interview** | Candidate is in the final interview stage |
| **Offer Extended** | A job offer has been sent |
| **Offer Accepted** | Candidate accepted the offer |
| **Offer Rejected** | Candidate declined the offer |
| **Hired** | Candidate has been hired |
| **Rejected** | Candidate was not selected |
| **Not Selected** | Not moved forward (softer than Rejected) |
| **Withdrawn** | Candidate withdrew their application |
| **Archived** | Application has been archived |

---

## Exporting Applicants

Click **Export CSV** to download the currently filtered applicant list. The export includes all visible columns and respects active filters.
