## Overview

The Assessment Review Dashboard gives you a detailed view of candidate performance, answer accuracy, time analytics, eye-tracking snapshots, and recommendation insights — helping you quickly evaluate candidates and make informed hiring decisions.

## Assessment Timeline

The timeline provides a complete overview of the assessment journey for each candidate, including:

- Assignment date and time
- Start time
- Completion status
- Assessment validity period

## Performance Metrics

| Metric | Description |
|---|---|
| **Overall Score** | Total percentage score achieved |
| **Section Scores** | Per-section breakdown for multi-test assessments |
| **Answer Accuracy** | Correct vs incorrect answers per question |
| **Time Analytics** | Time spent per question and per section |
| **Pass / Fail Status** | Based on the configured passing score |

---

## Proctoring Insights

If proctoring was enabled during the assessment, the review dashboard includes:

- Eye-tracking snapshots
- Tab-switching events
- Fullscreen violation logs
- Auto-submit events

<Note>
  Eye-tracking and behavioural analytics are used as supporting information only and should not be the sole basis for candidate evaluation.
</Note>

**Purpose of Eye Tracking:**
- Monitor candidate engagement
- Detect unusual movement patterns
- Improve proctoring and assessment integrity
- Provide additional behavioural insights

---

## Best Practices

- Use clear and meaningful assessment titles so candidates understand the purpose
- Configure realistic time limits and passing criteria based on question difficulty
- Review proctoring data alongside performance scores for a complete picture
- Use AI scores as an initial screening reference but always review responses manually
