## Overview

The Assessment Module helps you create, manage, assign, and monitor assessments efficiently. It provides quick access to test creation, assessment configuration, and result tracking.

<Note>
  Only users with **Admin** and **Editor** roles have permission to create assessments.
</Note>

## Dashboard Metrics

| Metric | Description |
|---|---|
| **Total Assessments** | Total number of assessments created |
| **Active Assessments** | Number of currently active assessments |
| **Drafts** | Assessments saved as drafts, not yet published |
| **Library Tests** | Tests available in the test library |

<Tip>
  Before creating an assessment, ensure at least one test exists in the Test Library. Create the test first — without published tests, assessment sections cannot be configured.
</Tip>

---

## Quick Actions

| Action | Description |
|---|---|
| **Create Assessment** | Build a reusable assessment template with sections and rules |
| **Browse Assessments** | View, edit, and manage existing assessment templates |
| **Create Test** | Create an aptitude or coding test for the library |
| **Browse Tests** | Track, manage, and reuse existing tests |

## Browsing Assessments

Search and filter all assessments created within the platform using the search bar in the top-right corner.

<Frame>
  <img src="/images/assessments/image59.png" alt="Browse assessments with filtering options" />
</Frame>

| Filter / Sort | Description |
|---|---|
| All / Draft / Published / Archived | Filter by status |
| Newest / Oldest First | Sort by creation date |
| Title A–Z / Z–A | Alphabetical sorting |
| Most / Least Questions | Sort by question count |

---

## Browsing Tests

Search for tests using the keyword or title search bar.

<Frame>
  <img src="/images/assessments/image60.png" alt="Browse tests with filtering options" />
</Frame>

| Filter / Sort | Description |
|---|---|
| All / Draft / Published / Archived | Filter by status |
| Newest / Oldest First | Sort by creation date |
| Recently Updated | Most recently modified tests |
| Most Questions / Most Marks | Tests with more questions / highest marks |
